Korkeatasoinen äänitysstudio soveltuu vaativaan musiikin äänittämiseen. Ollâtásásâš paddimstudio heivee vättee muusik paddiimân.
Studiotiloihin kuuluu studio, laulukoppi, äänitarkkaamo ja live-editointitila. Studiotiilijn kullojeh studio, lávlumvistááš, jienâtarkkuuttâh já live-editistemtile.
Studiotiloista vastaa Saamelaisalueen koulutuskeskus, mutta keskuksen ulkopuoliset varaukset tulevat keskitetysti Sajoksen Kongressi- ja tapahtumapalveluiden kautta. Studiotiilijn västid Säämi máttááttâskuávdáš, mutâ kuávdáá olgopiälásiih väärridmeh puátih Saijoos kongres- já tábáhtuspalvâlusâi peht.
Studio sijaitsee pääsisäänkäynnin vasemmalla puolella Saamelaisalueen koulutuskeskuksen media- ja kieliopiskelutilojen yhteydessä. Studio lii váldu-uuvsâ čižetbeln Säämi máttááttâskuávdáá media- ja kielâmáttááttâstiilij ohtâvuođâst.
Katso pohjapiirustus. Studio vuáđusárgus lii uáinimnáál vuáđusárgusist vyelemuu suájást.
Kuva: Tuupa Records / Jussi Isokoski ja Nillas Holmberg tekivät uuden levyn toukokuussa 2013. Kove: Tuupa Records / Jussi Isokoski já Nillas Holmberg raahtijn uđđâ skiäru vyesimáánust 2013.
Tutustu kyseisessä äänityksessä tehtyyn studiomusiikkivideoon. Uáppásm paddiimist rahtum studiomuusikvideon.
