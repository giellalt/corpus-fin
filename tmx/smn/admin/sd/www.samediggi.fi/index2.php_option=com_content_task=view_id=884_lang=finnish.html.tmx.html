Saamelaiskäräjät ilmoittaa haettavaksi koltansaamenkielisen oppimateriaalin projektityöntekijän Sämitigge almoot uuccâmnáál nuorttâlâškielâlii oppâmateriaal proojeektpargee
tehtävän ajalle 11.8.-31.12.2014. tooimâ ááigán 11.8.-31.12.2014.
Projektityöntekijän toimipaikka sijaitsee sihteeristön päätoimipaikassa saamelaiskulttuurikeskus Sajoksessa, Inarissa. Proojeektpargee toimâsaje lii čäällimkode váldutoimâsaajeest säämi kulttuurkuávdáš Sajosist, Anarist.
Projektityöntekijän tehtävänä on koltansaamenkielisen oppimateriaalin suunnittelu, laatiminen, kielen tarkistaminen ja käännösten valmistaminen. Proojeektpargee pargon lii nuorttâlâškielâlii oppâmateriaal vuávám, rähtim, kielâ tärhistem já jurgâlusâi rähtim.
