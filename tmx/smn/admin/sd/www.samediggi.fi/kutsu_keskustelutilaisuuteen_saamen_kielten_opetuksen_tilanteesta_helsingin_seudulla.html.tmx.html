Kutsu keskustelutilaisuuteen saamen kielten opetuksen tilanteesta Helsingin seudulla Povdiittâs savâstâllâmtilálâšvuotân sämikielâi máttááttâs tiileest Helsig kuávlust
Saamelaiskäräjät kutsuu Helsingin seudun vanhempia keskustelutilaisuuteen. Sämitigge puávdee Helsig kuávlu vaanhimijd savâstâllâmtilálâšvuotân.
Tilaisuudessa keskustellaan Helsingin seudun saamen kielten opetuksen tilanteesta sekä muista ajankohtaisista ja esille nousevista aiheista. Tilálâšvuođâst savâstâllâp Helsig kuávlu sämikielâi máttááttâs tiileest sehe eres äigikyevdilijn já oovdân puáttee fáádáin.
Paikalla ovat Saamelaiskäräjien puheenjohtaja Tuomas Aslak Juuso ja koulutussihteeri Bigga-Helena Magga. Pääihi alne lává Sämitige saavâjođetteijee Tuomas Aslak Juuso já škovlimčällee Bigga-Helena Magga.
Aika : 5.7.2023 kello 14.00-15.30 Äigi : 5.7.2023 tijme 14.00-15.30
Paikka : Microsoft teams, linkki keskustelutilaisuuteen Saje : Microsoft teams, liŋkkâ savâstâllâmtilálâšvuotân
Tilaisuus järjestetään etäyhteyksin Microsoft Teams-kokouksena. Tilálâšvuotâ uárnejuvvoo káidusohtâvuođâiguin Microsoft Teams-čuákkimin.
Toivomme, että mahdollisimman moni Helsingin seudun vanhempi pääsisi osallistumaan tilaisuuteen. Mij tuáivup, ete nuuvt maaŋgâ vaanhim, ko máhđulâš pesâččij servâđ tilálâšvuotân.
Mikäli tilaisuuteen liittymisessä on ongelmia, ole yhteydessä koulutussihteeriin. Jis tilálâšvuotân servâmist láá čuolmah, te vääldi ohtâvuođâ škovlimčällei.
