Katso lyhytvideo kolttasaamelaisesta lakista ! Keejâ uánihisvideo nuorttâsämmilâš kaperist
Kolttasaamelaisilla on neljä erilaista lakkimallia, joilla on erityinen merkitys. Nuorttâsämmilijn láá nelji sierâlágán kappeermaali, main lii eromâš merhâšume.
Kutakin lakkia saa käyttää vain tiettyyn siviilisäätyyn kuuluva henkilö. Jieškote-uv kappeer uážžu kevttiđ tuše tiätu siiviilsiätun kullee olmooš.
