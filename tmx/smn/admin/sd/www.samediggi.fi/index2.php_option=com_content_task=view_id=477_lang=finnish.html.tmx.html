Nuorisoneuvosto Nuorâirääđi
Nuorisoneuvostoon valitaan kahdeksi vuodeksi kerrallaan puheenjohtaja, varapuheenjohtaja ja kolme muuta jäsentä sekä jokaiselle henkilökohtaiset varajäsenet. Nuorâirááđán väljejuvvojeh kyevti ihán häävild saavâjođetteijee, värisaavâjođetteijee já kulmâ eres jeessân sehe juáhážân persovnliih värijesâneh.
Näiden varsinaisten jäsenten ja varajäsenten tulee olla valituksi tullessaan 18-25-vuotiaita. Taah eidusiih jesâneh já värijesâneh kalgeh leđe valjiidijn 18-25-ihásiih.
Lisäksi nuorisoneuvostoon valitaan kahdeksi vuodeksi kerrallaan viisi pysyvää asiantuntijajäsentä, joiden tulee olla valituksi tullessaan 15-17-vuotiaita. Lasseen nuorâirááđán väljejuvvojeh kyevti ihán häävild vittâ pissoo äššitobdeejeessân, kiäh kalgeh leđe valjiidijn 15-17-ihásiih.
Nuorisoneuvoston puheenjohtajan ja varapuheenjohtajan ei tarvitse olla saamelaiskäräjien jäseniä. Nuorâirääđi saavâjođetteijee já värisaavâjođetteijee iävá taarbâš leđe sämitige jesâneh.
Nuorisoneuvosto valmistelee ne saamelaiskäräjien lausunnot, aloitteet ja muut kannanotot jotka koskevat saamelaisnuoria tai saamelaisnuorten elinoloja. Nuorâirääđi valmâštâl taid sämitige ciälkkámušâid, alguid já eres peleväldimijd moh kyeskih säminuorâid tai säminuorâi iäláttâsâid.
Muita nuorisoneuvoston päätehtäviä on edistää saamelaisnuorten kielellisiä ja kulttuurisia oikeuksia ympäri Suomen sekä vahvistaa saamelaisnuorten kuuluvuutta saamelaiskulttuuriin. Nuorâirääđi eres váldupargoh láá ovdediđ säminuorâi kielâlâš já kulttuurlâš vuoigâdvuođâid pirrâ Suomâ sehe nanodiđ säminuorâi kulluuvâšvuođâ sämikulttuurân.
