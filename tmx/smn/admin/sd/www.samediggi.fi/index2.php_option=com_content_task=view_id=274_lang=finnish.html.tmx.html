Ilmoittautuminen Saamelaisnuorten taidetapahtumaan alkanut Almottâttâm Säminuorâi taaidâtábátusân lii álgám
Ilmoittautuminen Vuotsossa maaliskuun 26. päivänä järjestettävään Saamelaisnuorten taidetapahtumaan on alkanut. Almottâttâm Säminuorâi taaidâtábátusân, mii uárnejuvvoo Vuáčust njuhčâmáánu 26. peeivi, lii álgám.
Tapahtuman teemoina ovat teatteri ja lyhytelokuva. Tábáhtus teeman lii teatter já uánihiselleekove.
