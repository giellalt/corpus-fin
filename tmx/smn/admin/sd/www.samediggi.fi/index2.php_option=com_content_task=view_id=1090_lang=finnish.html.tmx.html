Saamelaiskäräjien sihteeristö on suljettu 6.-31.7.2015 Sämitige čälleetoimâttâh lii toppum 6.-31.7.2015
Saamelaiskäräjien sihteeristö on suljettu 6.-31.7.2015 välisen ajan. Sämitige čälleetoimâttâh lii toppum 6.-31.7.2015 koskâsâš ääigi.
Hallintopäällikön kesäloman sijaisena 1.-31.7.2015 toimii vs. lakimiessihteeri Kalle Varis, puh. 010 8393 187, gsm. 050 384 70 40. Haldâttâhhoovdâ kesiluámu sajasâžžân 1.-31.7.2015 tuáimá vs. lahâalmaičällee Kalle Varis, puh. 010 8393 187, gsm. 050 384 70 40.
