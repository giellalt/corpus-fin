Katso lyhytvideo Inarinsaamenkielen tilasta! Keejâ uánihisvideo aanaarsämikielâ tiileest
Räppäri Amoc alias Mikkâl Antti Morottaja tekee musiikkia inarinsaamen kielellä. Räppär Amoc alias Mikkâl Antti Morottaja ráhtá muusik aanaarsämikielân.
Amocin ja räpin kautta myös lapset kiinnostuvat kielen puhumisesta. Amoc ja räp pehti meid párnááh tobdeh mielâkivviivâšvuođâ kielâ sárnumân.
