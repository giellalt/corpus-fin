Ravintola vuokrattavana Raavâdviäsu laiguuttemnáál
Saamelaiskäräjät kilpailuttaa Saamelaiskulttuurikeskukseen tulevan ravintolan vuokrasopimuksen. Sämitigge kištoot Säämi kulttuurkuávdážân šaddee raavâdviäsu láigusopâmuš.
Inariin rakennettavan Saamelaiskulttuurikeskuksen on määrä aloittaa toimintansa vuonna 2012. Säämi kulttuur-kuávdás kalga algâttiđ tooimâs ive 2012.
Vuokralaisen kanssa tehdään toistaiseksi voimassa oleva vuokrasopimus ja vuokra-aika alkaa 1. tammikuuta 2012. Láigulâžžáin tahhoo vyeimist orroo láigusopâmuš tuáistáážân já láiguäigi álgá 1. uđđâivemáánu 2012.
Tarjoukset tulee toimittaa suljetussa kirjekuoressa liitteineen 14.8.2009 klo 16.00 mennessä Saamelaiskäräjien toimistoon Angelintie 696, 99870 Inari. Falâlduvâid lahtosijdiskuin kalga toimâttiđ toppum kirjekooverttist 14.8.2009 tme 16.00 räi Sämitige toimâttâhân Angelintie 696, 99870 Inari.
Kuoreen tulee merkitä viite ” Saamelaiskulttuurikeskuksen ravintola ”. Kooverttân kalga merkkiđ čujos ” Saamelais-kulttuurikeskuksen ravintola ”.
