Tilausravintola Raavâdviäsu Lyeme
Ravintolan isoista ikkunoista aukeaa upea näköala Juutuanjoelle ja osin myös Inarijärvelle. Lyeme-raavâdviäsu stuorrâ laasâi čoođâ lii fiijnâ uáinus Juvduujuuhân já uásild meid Aanaarjáávrán.
Sajoksen ravintola palvelee kongressi- ja tapahtuma-asiakkaita. Saijoos raavâdviäsu Lyeme palvâl kongres- já tábáhtusäššigâsâid, Saijoos pargovievâ já kuávdáá äššigâsâid.
Ravintolassa on 50 asiakaspaikkaa ja ravintolasalin vieressä oleva monitoimisali Dollagáddi on yhdistettävissä ravintolaan, jolloin istumapaikkoja on 140. Lyemest láá 60 äššigâssaijeed já maaŋgâtoimâsali Dollagáddi, mii lii raavâdviäsu paaldâst, puáhtá ovtâstittiđ raavâdviäsun, kuás čokkámsajeh láá 140.
Lataa ravintola Lyemen kokoustarjoiluhinnasto. KÄHVIVIÄSU
TILAUKSET/VARAUKSET: Raavâdviäsu Lyemest láá fáállun meid kähvi já liäibustuvah.
