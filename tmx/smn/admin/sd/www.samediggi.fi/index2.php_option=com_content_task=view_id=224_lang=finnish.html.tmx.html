Kolttalaulu Nuorttâlâšlaavlâ
Katso lyhytvideo kolttasaamelaisten perinteisestä laulutavasta ! Keejâ uánihisvideo nuorttâsämmilâš ärbivuáválii lávlumvyevist
Leudd eli kolttalaulu on perinteinen kolttasaamelainen laulutapa. LeuŠdd ađai nuorttâlâšlaavlâ lii ärbivuáválâš nuorttâsämmilij lávlumvyehi.
Leuddata voi mitä vain ja missä vain. LeuŠddiđ puáhtá maid peri já kost peri.
Filmissä tutustutaan Eliakseen sekä hänen lapsenlapseensa Hennaan ja heidän suhteeseensa leuddiin. Fiilmâst uápásmup Eljisân sehe suu áijubân Heenan já sunnuu koskâvuotân leuŠddân.
