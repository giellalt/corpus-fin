Tiedotustilaisuus Tieđettemtilálâšvuotâ
Senaatti-kiinteistöt ja Saamelaiskäräjät järjestävät tiedotustilaisuuden Saamelaiskulttuurikeskushankkeen tämän hetkisestä tilasta tiistaina 30.3.2010 kello 17.00 Siidan auditoriossa Inarin kirkonkylässä. Seenaat-kiddoduvah já Sämitigge orniv tieđettemtilálâšvuođâ Säämi kulttuurkuávdášhaavâ tááláá tiileest majebaargâ 30. peeivi njuhčâmáánu 2010 tijme 17.00 Siijdâ auditoriost Aanaar kirkkosiijdâst.
