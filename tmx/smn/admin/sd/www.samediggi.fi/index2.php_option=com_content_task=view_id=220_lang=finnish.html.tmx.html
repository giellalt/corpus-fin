Ante-pojan isä rakentaa joka talvi lumiteatteria Skábmagovat-elokuvafestivaalia varten. Ante-kaandâ eeči huksee jyehi täälvi muotâteatter Skammâkoveh-elleekovefestivaal várás.
Teatteri rakennetaan jäästä ja lumesta. Teatter huksejuvvoo jieŋâst já muottust.
Tällä kertaa Ante haluaa olla mukana rakennustöissä. Taan tove Ante haalijd leđe mieldi huksimpargoin.
