Skábmagovat alkuperäiskansojen elokuvafestivaalin lastenpäivän järjestelyt ja mediakasvatusmateriaalin kokoaminen. Skammâkoveh algâaalmugij elleekovefestivaal párnáipeeivi orniimeh já mediašoddâdemmateriaal nuurrâm.
