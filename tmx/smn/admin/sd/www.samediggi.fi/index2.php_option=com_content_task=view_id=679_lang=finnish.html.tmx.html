Inarinsaamen aapinen sai työkirjan Anarâškielâ aabis finnij pargokirje
Inarinsaamen vuonna 2011 julkaistu syyslukukauden aapinen on saanut rinnalleen työkirjan Anarâškielâlâš čohčâluuhâmpaje aabis, mii almostui ive 2011, lii tääl finnim olssis kyeimi
Työkirjan on tehnyt Matti Morottaja, Kuobža-Saammâl Matti, ja sen on kuvittanut Daniel Gelencsér. Pargokirje lii ráhtám Matti Morottaja, Kuobžâ-Saammâl Matti, já kuuvijd lii ráhtám Daniel Gelencsér.
Kirja on saatavissa Duodji shopista Sajoksesta ja saamelaiskäräjien koulutus- ja oppimateriaalitoimistosta, info (at) samediggi. Kirje finnee uástiđ Duodji - käävpist Sajosist já sämitige škovlim- já oppâmateriaaltoimâttuvâst, info (at) samediggi.
