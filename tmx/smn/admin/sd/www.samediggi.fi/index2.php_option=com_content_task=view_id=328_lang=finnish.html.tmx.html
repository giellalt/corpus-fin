Kutsu arkkitehtuurikilpailun tulosten julkistamistilaisuuteen 4.12.2008 Povdim arkkitehtkištodem puátusij almostittemtilálâšvuotân
Tervetuloa Saamelaiskulttuurikeskuksen arkkitehtuurikilpailun tulosten julkistamistilaisuuteen, joka pidetään Saamelaismuseo Siidan auditoriossa, Inarissa torstaina 4.12.2008 klo 15.
