Korkeatasoinen videostudio soveltuu monikamerajärjestelmän käyttöön ja muunkin videomateriaalin kuvaamiseen. Ollâtásásâš videostudio heivee maaŋgâkameravuáháduv kiävtun já eres-uv videomateriaal kuvviimân.
Studiotiloihin kuuluu studio, live-editointitila sekä muita editointitiloja. Studiotiilijd kullojeh studio, live-editistemtile sehe editistemtileh.
Studiotiloista vastaa Saamelaisalueen koulutuskeskus, mutta keskuksen ulkopuoliset varaukset tulevat Sajoksen Kongressi- ja tapahtumapalvelujen kautta. Studiotiilijn västid Säämi máttááttâskuávdáš, mutâ kuávdáá olgopiälásiih väärridmeh puátih Saijoos Kongres- já tábáhtuspalvâlusâi peht.
Studio sijaitsee Sajoksen ensimmäisessä kerroksessa, pääsisäänkäynnin vasemmalla puolella Saamelaisalueen koulutuskeskuksen media- ja kieliopiskelutilojen yhteydessä. Studio lii Saijoos vuossâmuu keerdist, váldu-uuvsâ čižetpeln Säämi máttááttâskuávdáá media- já kielâmáttááttâstiilij ohtâvuođâst.
