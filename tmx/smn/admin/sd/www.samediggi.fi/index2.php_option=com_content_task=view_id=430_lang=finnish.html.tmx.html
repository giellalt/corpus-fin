Saamelaiskäräjien kulttuuriavustusten hakeminen 2011 Sämitige kulttuurtorjui uuccâm 2011
Saamelaiskäräjät julistaa saamenkielisen kulttuurin edistämiseen ja saamelaisjärjetöjen toimintaan osoitetusta määrärahasta (Saamelainen kulttuurimääräraha) myönnettävät avustukset haettaviksi. Sämitigge almoot uuccâmnáál torjuid, moh mieđettuvvojeh sämikielâlâš kulttuur oovdedmân já sämiseervij tooimân čujottum meriruuđâst (Säämi kulttuurmeriruttâ).
Katso ilmoitus, lomakkeet ja hakuohjeet. Keejâ almottâs, luámáttuvah já uuccâmravvuuh.
