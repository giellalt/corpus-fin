Kutsu saamelaisnuorten konferenssiin Povdiittâs säminuorâi konfereensân
Tervetuloa Saamelaiskäräjien nuorisoneuvoston järjestämään saamelaisnuorten konferenssiin saamelaiskulttuurikeskus Sajokseen Inariin 6.-7.2.2014. Tiervâpuáttim Sämitige nuorâirääđi uárnim säminuorâi konfereensân säämi kulttuurkuávdáš Sajosân Anarân 6.-7.2.2014.
Konferenssin aiheena on suvaitsevaisuus sekä saamelaisnuorten identiteetti. Konferens fáddán lii soováášvuotâ sehe säminuorâi identiteet. Konferens eehidohjelmin lii sämimuusik.
Konferenssin iltaohjelmana on saamelaismusiikkia. Konferens lii mávsuttem já tot lii áávus puohháid kiäh láá perustum.
Konferenssi on maksuton ja se on avoin kaikille kiinnostuneille. TIERVÂPUÁTTIM !
Ilmoittautumiset viimeistään 24.1.2014. Almottâtmeh majemustáá 24.1.2014.
