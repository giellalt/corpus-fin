Kutsu saamelaisnuorten konferenssiin Povdiittâs säminuorâi konfereensân
Tervetuloa Saamelaiskäräjien nuorisoneuvoston järjestämään saamelaisnuorten konferenssiin saamelaiskulttuurikeskus Sajokseen Inariin 6.-7.2. Tiervâpuáttim Sämitige nuorâirääđi uárnim säminuorâi konfereensân säämi kulttuurkuávdáš Sajosân Anarân 6.-7.2.2014.
Konferenssin aiheena on suvaitsevaisuus sekä saamelaisnuorten identiteetti. Konferens fáddán lii soováášvuotâ sehe säminuorâi identiteet.
Konferenssin iltaohjelmana on saamelaismusiikkia. Konferens eehidohjelmin lii sämimuusik.
Konferenssi on maksuton ja se on avoin kaikille kiinnostuneille. Konferens lii mávsuttem já tot lii áávus puohháid kiäh láá perustum.
Ilmoittautumiset viimeistään 24.1. Almottâtmeh majemustáá 24.1.2014.
