Projektit Projekteh
Täältä voit käydä katsomassa Saamelaiskäräjien käynnissä olevia projekteja. Tääbbin puávtáh eelliđ keččâmin projektijd, moh láá Sämitiggeest joođoost.
Päättyneiden projektin loppuraportit löydät dokumenttipankista PROJEKTIT-kansiosta. Nuuhâm projektij loppâraportijd kaavnah dokumentpaaŋkist PROJEKTEH-maapist.
