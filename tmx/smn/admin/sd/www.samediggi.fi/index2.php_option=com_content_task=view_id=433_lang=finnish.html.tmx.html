Saamelaiskäräjävaalit 2011 Sämitiggevaaljah 2011
Saamelaiskäräjien seuraavat vaalit toimitetaan syyskuussa 2011. Sämitige puáttee vaaljah toimâttuvvojeh čohčâmáánust 2011.
Äänioikeutettuja vaaleissa ovat kaikki ne saamelaiset, jotka on merkitty saamelaiskäräjien vaaliluetteloon. Jienâstemvuoigâdvuođâliih vaaljâin láá puoh toh sämmiliih, kiäh láá merkkejum sämitige vaaljâluvâttâlmân.
