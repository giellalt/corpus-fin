Saamelaiskulttuurikeskuksen rakennustyö alkoi vanhojen rakennusten purkauksella Säämi kulttuurkuávdáá huksimpargo aalgij 12.4. Anarist puári
Saamelaiskulttuurikeskuksen rakennustyö alkoi maanantaina 12.4.2010. Säämi kulttuurkuávdáá huksimpargo aalgij vuossaargâ 12.4.2010.
Työmaa aloitettiin purkamalla keskuksen tontilta vanhat rakennukset pois. Pargo algâttui raččomáin puáris rakânâsâid kuávdáá toontist meddâl.
Purku-urakointi kestää viikolle 18/2010. Raččomurkkodem pištá ookon 18/2010.
Rakennustyömaan urakoinnista vastaa Senaatti-kiinteistöt. Huksimpargo urkkodem kulá Senaatti-kiddoduvváid.
