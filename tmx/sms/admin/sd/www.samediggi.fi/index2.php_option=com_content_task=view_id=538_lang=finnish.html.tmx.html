Sajos Tuåimmpäiʹǩǩteevvai ââʹnn huõl vahssimäästarkääzzkõõzzi, infopääiʹǩ håiddmõʹšše, såbbri da noorõõttâmpooddi riâššmõʹšše kuulli tuâjain di taarbšeʹmmen čiistmõõžžâst.
