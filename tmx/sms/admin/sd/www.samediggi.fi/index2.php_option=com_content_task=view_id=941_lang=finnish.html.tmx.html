Saamelaiskäräjät julistaa haettavaksi yhteensä kuusi 500 euron stipendiä, jotka on tarkoitettu saamen kielen ja/tai saamelaisen kulttuurin pääaine- ja sivuaineopiskelijoille korkea-asteen koulutuksessa.
