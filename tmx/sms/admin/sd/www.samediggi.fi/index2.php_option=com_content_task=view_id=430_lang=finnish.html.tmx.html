Saamelaiskäräjät julistaa saamenkielisen kulttuurin edistämiseen ja saamelaisjärjetöjen toimintaan osoitetusta määrärahasta (Saamelainen kulttuurimääräraha) myönnettävät avustukset haettaviksi.
