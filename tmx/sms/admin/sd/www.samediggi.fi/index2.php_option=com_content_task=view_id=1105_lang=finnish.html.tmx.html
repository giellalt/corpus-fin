Saamelaiskäräjät ilmoittaa haettavaksi saamen kielen toimiston vakinaisen koltaasaamen kielenkääntäjän viran. Sääʹmteʹǧǧ iʹlmmat ooccâmnalla Sääʹm ǩiõl konttâr põõšši nuõrtts ääʹm ǩiõlljåårǥlõ ʹtti vee ʹrj.
jonka toimipaikka sijaitsee saamelaiskäräjien sihteeristössä saamelaiskulttuurikeskus Sajoksessa Inarin kirkonkylässä. koon tuåimmpäiʹǩǩ sâʹjjdââtt sääʹmteeʹǧǧ piisarkådda sääʹmkulttuurkõõskõs Sajoʹsse Aanar ceerkavsiidâst.
