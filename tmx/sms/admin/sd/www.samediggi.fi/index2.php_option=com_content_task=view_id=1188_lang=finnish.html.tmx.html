Saamelaiskäräjät ilmoittaa haettavaksi koltansaamenkielisen oppimateriaalin projektityöntekijän tehtävän ajalle 1.4. - 31.12.2016. Sääʹmteʹǧǧ iʹlmmat ooccâmnalla nuõrttsääʹmǩiõllsa mättmateriaal projeʹktt-tuâjjla tuâi ääiǥas 1.4. - 31.12.2016.
