Saamelainen parlamentaarinen neuvosto vaati vetoomuksessaan 23.9.2011 välitöntä toimenpidekieltoa YK:n ihmisoikeusneuvostolta Nellimin siidan eli tokkakunnan porojen pakkoteurastusten estämiseksi.
