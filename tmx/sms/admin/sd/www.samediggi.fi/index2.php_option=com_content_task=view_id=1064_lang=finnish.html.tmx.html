Saamelaiskäräjät antoi stipendin neljälle Saamelaisalueen koulutuskeskuksesta valmistuneelle ahkeralle saamen kielen opiskelijalle. Sääʹmteʹǧǧ jueʹjji stipeeʹndid nellja Sääʹmvuuʹd škoouʹlʼjemkõõskõõzzâst valmštõõvvi reʹttev sääʹmǩiõl mättʹtõõttja.
Saamelaisalueen koulutuskeskuksen kevätjuhlaa vietettiin keskiviikkona kesäkuun kolmantena päivänä. Sääʹmvuuʹd škoouʹlʼjemkõõskõõzz ǩiđđprääʹzneǩ prääzkješ seärad ǩieʹssmannu kuälmad peeiʹv.
