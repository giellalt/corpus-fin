Saamelaiskäräjät jakoi stipendejä saamen kielen oppilaille Sámediggi attii stipeanddaid sámegiela oahppiide
Saamelaiskäräjät jakoi koulujen kevätjuhlissa stipendejä saamen kielen opinnoissa hyvin menestyneille oppilaille. Sámediggi attii skuvllaid giđđafeasttain stipeanddaid oahppiide, geat leat bures menestuvvan sámegiela oahpuin.
Stipendejä jaettiin yhteensä seitsemälle saamen kielen kokeen ylioppilastutkinnossa suorittaneelle ja 19:lle perusopetuksen päättäneelle oppilaalle.
