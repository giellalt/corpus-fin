Projektit Prošeavttat
Täältä voit käydä katsomassa Saamelaiskäräjien käynnissä olevia projekteja. Dáppe gávnnat Sámedikki proševttaid.
Päättyneiden projektin loppuraportit löydät dokumenttipankista PROJEKTIT-kansiosta. Čađahuvvon proševttaid loahparaportaid gávnnat dokumeantabáŋkku PROŠEAVTTAT-máhpas.
