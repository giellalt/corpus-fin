Saamelaiskulttuurikeskuksen arkkitehtuurikilpailu käynnistynyt Arkitektuvragilvvu Sámekulturguovddáža plánemis
Senaatti-kiinteistöt järjestää yleisen, kaksivaiheisen arkkitehtuurikilpailun Saamelaiskulttuurikeskuksen suunnittelusta. Senáhtta-giddodagat ordne almmolaš, guovtteoasát arkitektuvragilvvu Sámekulturguovddáža plánemis.
Ensimmäinen vaihe alkaa 14.4.2008 ja päättyy 30.6.2008. Vuosttas muddu álgá 14.4.2008 ja nohká 30.6. 2008.
Kilpailun toinen vaihe alkaa 1.9.2008 ja päättyy 3.10.2008. Gilvvu nubbi muddu álgá 1.9.2008 ja nohká 3.10.2008.
