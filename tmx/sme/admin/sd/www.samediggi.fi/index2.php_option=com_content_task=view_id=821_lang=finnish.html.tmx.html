Uusi nuorisoneuvosto on asetettu Uumajassa Ođđa nuoraidráđđi lea ásahuvvon Upmis
Saamelaiskäräjien kokous asetti nuorisoneuvoston toimikaudeksi 2014-2015 kokouksessaan Uumajassa 19.2.2014. Sámedikki čoahkkin ásahii nuoraidráđi doaibmaáigodahkii 2014-2015 čoahkkimisttis Upmis 19.2.2014.
Nuorisoneuvostoa edusti kokouksessa Niila Rahko. Nuoraidráđi lei ovttasteamen čoahkkimis Niila Rahko.
