Määräaikainen toimistosihteeri Mearreáigásaš doaimmahatčálli
Saamelaiskäräjät hakee yhteispohjoismaisen saamenkielen ammatti- ja resurssikeskuksen perustamishankkeeseen Sámediggi ohcá davviriikkalaš sámegiela fága- ja resursaguovddáža ásahanprošektii mearreáigásaš doaimmahatčálli áigodahkii 1.1.2013-30.6.2014.
