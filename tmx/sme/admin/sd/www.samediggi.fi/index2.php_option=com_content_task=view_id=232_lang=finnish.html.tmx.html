Katso lyhytvideo kolttasaamelaisesta lakista ! Geahča oanehisvideo nuortalaččaid gahpira birra
Kolttasaamelaisilla on neljä erilaista lakkimallia, joilla on erityinen merkitys. Nuortalaččain leat njeallje sierralágan gahpermálle, main lea erenomáš mearkkašupmi.
Kutakin lakkia saa käyttää vain tiettyyn siviilisäätyyn kuuluva henkilö. Guđege gahpira oažžu geavahit dušše dihto siviilasehtui gullevaš olmmoš.
