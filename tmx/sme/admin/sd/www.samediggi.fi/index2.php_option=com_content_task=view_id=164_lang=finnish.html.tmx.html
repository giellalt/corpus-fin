Saamelaiskulttuurikeskuksen arkkitehtuurikilpailun kilpailutyöt Sámekulturguovddáža arkitektuvragilvvu barggut
Saamelaiskulttuurikeskuksen arkkitehtuurikilpailun ensimmäinen vaihe päättyi 30.6.2008. Sámekulturguovddáža arkitektuvragilvvohallama vuosttaš muddu nogai 30.6.2008.
