Beaivvá Sámi Teahterin näytelmä 24.9.2013

Beaivváš Sámi Našunálateáhter esitää "Ridn´oaivi ja nieguid oaidni ("Kuurapää ja unien näkijä") 24.9.2013 Sajoksen Dolla-salissa. Áillohaš olisi täyttänyt tänä vuonna 70 vuotta ja tässä yhteydessä Beaivváš Teahter näyttää hänen ainoan kirjoittamansa näytelmän. Beaivváš kulkee ympäri maailmaa tämän esityksen kanssa. Tiistaina 24.9.2013 he tulevat Inariin, Sajokseen.

 

Lipunmyynti alkaa tuntia ennen esityksen alkua ovelta. Liput 20€ aikuiset/10€ opiskelijat. Ryhmälippu -20%. Järjestäjä: Beaivváš Sámi Našunálateáhter. Miniohjelma myös suomen kielellä

Viimeksi päivitetty ( 19.08.2013 )
