Inarinsaamen kielenkääntäjän virka
Saamelaiskäräjät julistaa haettavaksi saamen kielen toimiston vakinaisen Inarinsaamen kielenkääntäjän viran, jonka toimipaikka sijaitsee saamelaiskäräjien sihteeristössä saamelaiskulttuurikeskus Sajoksessa Inarin kirkonkylässä.
Viimeksi päivitetty ( 07.11.2012 )
