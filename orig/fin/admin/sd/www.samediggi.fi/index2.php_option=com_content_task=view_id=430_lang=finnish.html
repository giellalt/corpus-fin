Saamelaiskäräjien kulttuuriavustusten hakeminen 2011
Saamelaiskäräjät julistaa saamenkielisen kulttuurin edistämiseen ja saamelaisjärjetöjen toimintaan osoitetusta määrärahasta (Saamelainen kulttuurimääräraha) myönnettävät avustukset haettaviksi. Katso ilmoitus, lomakkeet ja hakuohjeet. 
Viimeksi päivitetty ( 17.11.2010 )
