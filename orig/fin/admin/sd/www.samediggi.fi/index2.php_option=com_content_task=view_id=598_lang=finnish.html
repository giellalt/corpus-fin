Saamelaiskäräjät jakoi stipendejä saamen kielen oppilaille
Saamelaiskäräjät jakoi koulujen kevätjuhlissa stipendejä saamen kielen opinnoissa hyvin menestyneille oppilaille. Stipendi annettiin yhteensä 13:lle saamen kielen kokeen ylioppilastutkinnossa suorittaneelle ja 18:lle perusopetuksen päättäneelle.

 

Saamelaiskäräjien stipendin saivat seuraavat nuoret:
