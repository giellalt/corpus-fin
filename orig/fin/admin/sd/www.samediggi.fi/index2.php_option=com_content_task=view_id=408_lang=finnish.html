Kongressi- ja tapahtumapalvelut

Saamelaiskulttuurikeskuksen asiakaspalveluyksikön, Kongressi- ja tapahtumapalvelujen tehtävänä on vastata keskuksen yhteistilojen markkinoinnista, myynnistä ja varauksista, huolehtia tapahtuma- ja kokousjärjestelyistä sekä hoitaa tekniikka- informaatio- ja vahtimestaripalveluita.

 

Lisätietoa saamelaiskulttuurikeskus Sajoksen vuokrattavista tiloista saat Sajoksen kotisivujen Järjestäjälle -sivulta.

 

  

Viimeksi päivitetty ( 19.09.2014 )
