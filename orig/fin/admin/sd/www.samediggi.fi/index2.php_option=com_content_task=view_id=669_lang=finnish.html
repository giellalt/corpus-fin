Saamelaisnuorten taidetapahtuma 2013 järjestetään Hetassa

Seuraava Saamelaisnuorten taidetapahtuma (Sámenuoraid dáiddadáhpáhus) järjestetään Marianpäiväviikolla torstaina 14.3. Hetan koulukeskuksessa Enontekiöllä. Tapahtuman teemana on teatteri. Hetassa valitaan saamelaisnuorten edustajat Helsingissä toukokuussa pidettävään Nuori Kulttuuri TEATRIS 2013 -tapahtumaan.

 

Saamelaiskäräjillä tapahtuman järjestelyistä vastaa suunnittelija Anni Näkkäläjärvi. Seuraa nettisivuja - toimitamme lisätietoja taidetapahtumasta mahdollisimman pian. Tapahtuman paikasta päätti koulutus- ja oppimateriaalilautakunta kokouksessaan 17.12.

 

Viimeksi päivitetty ( 18.12.2012 )
