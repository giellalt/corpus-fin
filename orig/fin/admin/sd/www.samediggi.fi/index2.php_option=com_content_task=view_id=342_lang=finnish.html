Ensimmäinen pohjoissaamenkielinen koulukartasto ilmestyi
Ensimmäinen saamenkielinen koulukartasto on ilmestynyt. ´Skuvlla kártagirji´ on oppilaan peruskartasto ja se sisältää tietoa Suomesta, Pohjoismaista ja eri maanosista. Kartasto sisältää myös luettelon itsenäisistä valtioista lippuineen ja saamenkielisen hakemiston. Lue lisää tästä.
Viimeksi päivitetty ( 12.11.2009 )
