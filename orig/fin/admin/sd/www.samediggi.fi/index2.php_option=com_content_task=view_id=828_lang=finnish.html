Tiedote
Saamelaiskäräjät ottaa käyttöön uuden sähköpostijärjestelmän viikolla 11. Järjestelmän käyttöönotto aloitetaan maanantaina 10.3.2014. Järjestelmän käyttöönoton aikana sähköpostiliikenteessä voi olla viivästyksiä. Kiireellisissä asioissa kehotamme ottamaan yhteyttä saamelaiskäräjiin puhelimitse . 
Viimeksi päivitetty ( 15.03.2014 )
