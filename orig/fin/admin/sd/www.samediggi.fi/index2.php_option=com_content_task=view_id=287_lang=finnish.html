Anna Näkkäläjärvestä vs. kulttuurisihteeri

Saamelaiskäräjien yleisessä toimistossa haettavana olleen kulttuurisihteerin viransijaiseksi on otettu musiikin maisteri Anna Näkkäläjärvi. Näkkäläjärvi aloittaa tehtävässään 18. toukokuuta 2009. Hänen toimipaikkansa on Saamelaiskäräjien yleisessä toimistossa Inarin Riutulassa. Viransijaisuutta määräaikaan mennessä haki Anna Näkkäläjärven lisäksi neljä muuta.

 

Lisätietoja hallintopäällikkö Juha Guttomilta. 

Viimeksi päivitetty ( 14.05.2009 )
