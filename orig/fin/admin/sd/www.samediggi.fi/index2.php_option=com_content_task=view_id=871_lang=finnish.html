Saamelaiskäräjät antoi stipendejä Saamelaisalueen koulutuskeskuksesta valmistuneille
Saamelaiskäräjät antoi stipendin kuudelle Saamelaisalueen koulutuskeskuksesta valmistuneelle ahkeralle saamen kielen opiskelijalle. Kolme heistä suoritti koltansaamen ylioppilaskokeen kevään ylioppilaskirjoituksissa. Saamelaisalueen koulutuskeskuksen kevätjuhlaa vietettiin keskiviikkona kesäkuun neljäntenä päivänä.

Saamelaiskäräjien stipendin saivat seuraavat opiskelijat:

Pohjoissaame: Juulienna Näkkäläjärvi

Inarinsaame: Henna Aikio

Koltansaame: Jarno Mäki

 

Koltansaamen ylioppilaskokeen kevään kirjoituksissa suorittaneet:

Minna Lehtola, Heidi Gauriloff ja Hannu Pessa.

 

Saamelaiskäräjät toivottaa onnea ja hyvää kesälomaa kaikille!  
Viimeksi päivitetty ( 04.06.2014 )
