Eduskuntavaalit pidetään joka neljäs vuosi

Vaaleilla valitaan 200 kansanedustajaa päättämään yhteisistä asioistamme. Vuoden 2011 eduskuntavaalit järjestetään sunnuntaina 17.4.2011. Lisätietoja www.vaalit.fi -sivulta. Tietoa eduskuntavaaleista myös pohjois-, inarin- ja koltansaamenkielillä.

Viimeksi päivitetty ( 24.03.2011 )
