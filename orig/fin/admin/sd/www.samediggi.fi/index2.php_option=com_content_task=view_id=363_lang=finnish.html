Saamelaiskulttuurikeskuksen rakennustyö alkoi vanhojen rakennusten purkauksella

Saamelaiskulttuurikeskuksen rakennustyö alkoi maanantaina 12.4.2010. Työmaa aloitettiin purkamalla keskuksen tontilta vanhat rakennukset pois. Purku-urakointi kestää viikolle 18/2010. Rakennustyömaan urakoinnista vastaa Senaatti-kiinteistöt.

 

 

Viimeksi päivitetty ( 25.07.2010 )
