Saamelaiskäräjät hakee kielipesäohjaajaa määräaikaiseen virkasuhteeseen

Saamelaiskäräjät hakee kielipesäohjaajaa määräaikaiseen virkasuhteeseen. Kielipesäohjaajan tehtäviin kuuluu mm. saamelaisten kielipesätoiminnan ohjaaminen, koordinointi ja kehittäminen saamelaisalueella ja sen ulkopuolella sekä kielipesien henkilöstöhallinto. Hakemus liitteineen tulee toimittaa Saamelaiskäräjien sihteeristöön 8.2.2016 klo 16.00 mennessä osoitteella Sajos, 99870 Inari.

Viimeksi päivitetty ( 25.01.2016 )
