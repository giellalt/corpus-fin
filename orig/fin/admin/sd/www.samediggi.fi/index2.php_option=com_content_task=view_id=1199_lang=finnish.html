Saamelaiskäräjät palkkaa kesätyöntekijöitä oppimateriaalitehtäviin
Saamelaiskäräjät palkkaa kesätyöntekijöitä oppimateriaalitehtäviin. Vapaamuotoinen hakemus työkokemus- ja opiskelutodistuksin lähetetään Saamelaiskäräjien koulutus- ja oppimateriaalilautakunnalle helmikuun 29. päivään 2016 klo 16 mennessä. Työ kestää 1 - 3 kuukautta, työn alkamisesta sovitaan erikseen. Hakemuksen voi lähettää sähköpostilla osoitteeseen Tämä sähköpostiosoite on suojattu roskapostia vastaan, aseta javascripttuki päälle nähdäksesi osoitteen. tai postiosoitteeseen: Saamelaiskäräjät/ Koulutus- ja oppimateriaalitoimisto, Sajos, 99870 Inari. 
Viimeksi päivitetty ( 14.02.2016 )
