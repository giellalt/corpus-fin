Saamelaisnuorten taidetapahtuma 2010 Rovaniemellä
Saamelaisnuorten taidetapahtuma 2010 järjestetään Rovaniemellä kaksipäiväisenä tapahtumana maaliskuussa. Saamelaiskäräjien koulutus- ja oppimateriaalitoimisto aloittaa parhaillaan taidetapahtuman järjestelyitä yhteistyössä Rovaniemen kaupungin kanssa. Luvassa on värikäs saamelaisnuorten kulttuuri- ja taideharrastusten juhla! Seuraa nettiuutisia www.samediggi.fi .  
Viimeksi päivitetty ( 27.10.2009 )
