Cokka fal ná

Katso lyhytvideo saamenpuvun pukemisesta!

 

Ingá, Ánne ja Niiles pääsevät ripille ja pukevat saamenpuvut päälleen. Pukeminen ei kuitenkaan ole kovin helppoa, vaan tarkkaa työtä, johon tarvitaan apua.

 

 

 

Päähenkilöt: Ingá ja Ánne Koivisto, Niiles Holmberg, Kirsti Paadar

Ohjaus/kuvaus: Ima Aikio-Arianaick, leikkaus: Anne Kirste Aikio, Tero Titoff

Musiikki: Pentti Kusmin, käännös: Birit Vuolab

Tuotanto: Sámi Siida Ohcejohka 2005

Pituus: 3,11

 

 

Viimeksi päivitetty ( 11.11.2013 )
