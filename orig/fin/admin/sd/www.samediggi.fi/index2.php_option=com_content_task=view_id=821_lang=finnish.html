Uusi nuorisoneuvosto on asetettu Uumajassa

Saamelaiskäräjien kokous asetti nuorisoneuvoston toimikaudeksi 2014-2015 kokouksessaan Uumajassa 19.2.2014. Nuorisoneuvostoa edusti kokouksessa Niila Rahko.
