Sofia Jannok ja Nanook esiintyvät Ijahis Idjassa Sajoksessa
Ijahis idja näyttää tänä kesänä entistä kansainvälisemmän puolensa. Festivaalin pääesiintyjäksi on saatu saamelaisen popin prinsessa Sofia Jannok Ruotsista. Kaukaisimmat vieraat tänä vuonna saapuvat Grönlannista. Ijahis idjassa esiintyy rock-yhtye Nanook joka esittää laulunsa grönlannin kielellä. Lisätietoa Ijahis Idjan ohjelmasta http://www.ijahisidja.fi/.  Lue lisää.
Viimeksi päivitetty ( 14.08.2012 )
