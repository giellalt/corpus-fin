Saamelaiskäräjät kehittää kotisivujaan
Saamelaiskäräjät on aloittanut kotisivujensa uudistamisen. Kotisivuille on lisätty uutta tietoa saamelaisista lainsäädännössä, saamelaiskäräjien edustajista eri työryhmissä , alkuperäiskansojen oikeuksia käsittelevistä kansainvälisistä sopimuksista  sekä alkuperäiskansojen oikeuksien toteutumista valvovien elimien uusimmista suosituksista Suomelle. Uusia sivuosioita käännetään parhaillaan saameksi. 
Viimeksi päivitetty ( 15.01.2014 )
