Saamelaiskäräjien vaalien ehdokasasettelu päättyy 5.8.2011

Ehdokkaan saamelaiskäräjien vaaleihin voi asettaa vähintään kolme äänioikeutettua saamelaista jättämällä vaalilautakunnalle asianmukaisesti täytetyn ja allekirjoitetun ehdokashakemuksen viimeistään 5.8.2011 ennen kello 16.

Hakemuslomakkeita on tulostettavissa Saamelaiskäräjien www-samediggi.fi sivuilta. Lomakkeita voi myös pyytää vaalilautakunnan toimistosta puhelin 010 839 3143 tai sähköpostitse Tämä sähköpostiosoite on suojattu roskapostia vastaan, aseta javascripttuki päälle nähdäksesi osoitteen.

Viimeksi päivitetty ( 08.08.2011 )
