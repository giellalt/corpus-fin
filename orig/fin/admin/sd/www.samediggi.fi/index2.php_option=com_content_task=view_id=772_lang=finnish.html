Saamelaiskäräjät hakee määräaikaista projektisihteeriä

Saamelaiskäräjät hakee määräaikaista projektisihteeriä Sáfá2- projektiin (Sámi Giellagáldu). Työ alkaa 16.12.2013 ja kestää 30.6.2014 saakka. Projektisihteerin tehtäviin kuuluvat projektin päivittäiset toimistotehtävät, kokousten järjestäminen sekä projektin kirjanpitotehtävistä huolehtiminen.

Viimeksi päivitetty ( 10.01.2014 )
